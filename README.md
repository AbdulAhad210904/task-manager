# Task Manager

**Task Manager** is a full-stack web application designed for efficient task management. It supports hierarchical parent-child tasks, automatic archiving of completed tasks, and paginated views. Built with React, Node.js, and MongoDB, it offers a modern, responsive user experience for organizing and managing large task lists.


### 2. Configure Environment Variables

Create a `.env` file inside the `backend/` folder with the following content:

```env
PORT=5000
MONGODB_URI=
```

### 3. Start the Application

#### Backend Server:

```bash
cd task-manager-backend
npm start
```

#### Frontend Development Server:

```bash
npm run dev
```
